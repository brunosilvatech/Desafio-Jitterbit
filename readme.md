# Order API

API simples para gerenciamento de pedidos utilizando Node.js e Express.

## Tecnologias usadas
- Node.js
- Express

## Como instalar

1. Clonar o projeto
2. Instalar dependências

npm install

## Como rodar o projeto

node server.js

O servidor rodará em:
http://localhost:3000

## Rotas da API

Criar pedido:
POST /order

Listar pedidos:
GET /order/list

Buscar pedido:
GET /order/:id

Deletar pedido:
DELETE /order/:id
