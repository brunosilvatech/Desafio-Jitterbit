# API de Pedidos

API desenvolvida em Node.js com Express e MongoDB para gerenciamento de pedidos.

## Endpoints

Criar pedido
POST
http://localhost:3000/order

Buscar pedido
GET
http://localhost:3000/order/:id

Listar pedidos
GET
http://localhost:3000/order/list

Atualizar pedido
PUT
http://localhost:3000/order/:id

Deletar pedido
DELETE
http://localhost:3000/order/:id
